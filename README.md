key text editor

<br>

Is a simple text editor written in C using ncurses.

`Key` is based in the [vim](https://www.vim.org/) text editor using some keybindings, for example: to move around uses `h`, `j`, `k`, `l` and some others.

Compile:

- Dependencies:
    - `ncurses - 6.6.20251230`
    - `gcc - 14.2.1`
    - `makefile`

Just run `make` in root directory.

To execute use `make run`.
